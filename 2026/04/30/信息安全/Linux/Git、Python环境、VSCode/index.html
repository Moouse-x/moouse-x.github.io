Linux下git的安装与使用

ubuntu下

  1. git的安装
    sudo apt update sudo apt intstall git git –version`
    分别是更新软件包列表
    安装git
    验证安装
  2. 配置git基本信息
    git config --global --list
    看是否有用户名和邮箱，若无则继续
    git config --global user.name "Name"
    git config --global user.email "email@example.com"
    然后验证配置git config --list
    或者直接查看配置文件cat ~/.gitconfig
  3. 使用git
    mkdir git_test
    cd git_test
    git init
    这是创建文件夹并初始化git存储库
    git status
    将文件放入git_test文件夹，查看状态。当新增文件或更改了当前文件，即可提交到git存储库
    git add .
    git commit -m "first commit"
    git log
    提交完以后查看提交日志，说明提交到本地仓库成功。
