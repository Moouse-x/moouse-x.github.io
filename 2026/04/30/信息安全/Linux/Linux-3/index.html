日期相关

date - 显示当前时间
基本语法
date +%Y显示当前年份
date +%m显示当前月份
date +%d显示当前是哪一天
date “+%Y-%m-%d %H:%M:%S”
date -d ‘1 days ago’–显示前一天的时间
date -d ‘-1 days ago’–显示明天时间
设置系统时间
date -s 字符串时间
cal 选项 日月年 - 日历
选项：
-1 只显示当前月份
-3 显示上月、当月和下月
-s 周日作为一周的第一天
-m 周一作为一周的第一天
-j 显示每一天是本年的第几天
-y 输出整年
-v 显示版本信息并退出
-h 显示此帮助并退出

进程相关命令

ps命令
ps -aux
-a 选择所有进程

shell编程

在Linux操作系统上编写一段脚本，但操作系统不能直接运行，需要借助解释器工具把脚本解析成为机器语言，操作系统就可以执行
Linux默认的shell解释器是bash

基础语法

shell脚本
格式：以#!/bin/bash开头
执行：
使用bash执行
获取执行权限：chmod 744 01.sh
变量
常用系统变量
$HOME
$PATH
$USER
$PWD
$SHELL…
set查看shell中的变量
自定义变量
变量=值
撤销变量 unset 变量
声明静态变量 readonly 变量 （该变量不能unset）
规则：
变量名称可以由字母、数字和下划线组成，不能由数字开头，环境变量名建议大写
等号两侧不能有空格
变量默认类型都是字符串类型，无法直接进行数值运算
变量的值如果有空格，需要使用双引号或单引号括起来
==read读取控制台输入==
read 选项 参数
选项：
-p 指定读取值时的提示符
-t 指定读取值时等待的时间（秒）

export 变量
