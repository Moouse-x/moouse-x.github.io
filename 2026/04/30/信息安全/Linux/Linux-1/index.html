Linux基础命令

#linux命令
ls +[-al] +[dir ]

pwd 打印当前所属的目录的路径

cd
cd .. 切换到上级目录
cd ~切换到home目录
cd /usr/local 切换具体目录
cd app 切换到app目录
cd /切换到根目录
cd - 回退上一个命令

touch 1.txt -创建空文件

cat [-n] filename -n可展示行数，查看文件内容

more filename 以分页形式显示文件内容

tail 默认查看文件末尾10 行 tail -20 /etc/profile 显示/etc目录下profile的末尾20行
例如：==动态查询日志==：tail -f catalina.out (在logs 文件夹)

head 查看文件前10行

mkdir [-p] dirname -p 递归创建目录

rmdir [-p] dirname -p 删除==空==目录

rm [-rf] name -r 递归删除 -f删除的时候不需要提示 –删除文件或者目录 rm -rf /* 删除得干干净净

cp [-r] source dest -r用于复制该目录下的所有子目录和文件 –用于复制文件或目录

mv source dest 为文件或目录改名，将文件或目录移动到其他位置
例如 mv 01.txt /usr/local

#linux命令/tar解压缩
tar 压缩和解压缩
-z z代表gzip 通过gzip对文件压缩或解压
-c c代表create 创建新的包文件
-v v代表verbose 用于显示命令的执行过程
-f f代表file 用于指定包文件的名称
-x x代表extract 实现从包文件中还原文件
常见的解压缩： tar -xvf nginx-1.24.0.tar.gz -C /usr/local

vi和vim 命令
vi filename - 命令模式查看文件
i-插入模式 编辑文本 Esc-退出插入模式
:-底行模式 :wq-保存并退出
