SQL(Structed Query Language)

SQL语句

SQL分类

SQL分为数据定义语言(DDL),数据操纵语言(DML),事务控制语言(TCL),数据控制语言(DCL)

数据定义语言(Data Definition Language)
create alert drop truncate
数据操纵语言(Data Manipulation Language)
insert update delete select
事务控制语言(Transaction Control Language)
commit savepoint rollback
数据控制语言(Data Control Language)
grant revoke

Oracle数据类型

字符型：
char 固定长度(1-2000字节)
varchar2 可变长度(1-4000字节)
long 可变长度(2GB)

数值类型：
number: 整数如：number(3)；小数如:numer(5,2) 其中5为精度，2为小数位数

时间类型：
date:存储日期和时间，精确到秒
timestamp:同上存储，还存储时区信息

二进制数据类型：RAW(2000字节)与LANG RAW(2GB)
Lob数据类型：LOB(4GB)

Oracle表管理、约束等

创建表：create table 表名
修改表：alter teble 表名 add(新增列)/modify(修改列)/drop(删除列) [column]/rename(修改列名) [column] to ;
删除表：drop table 表名

表约束
  1. 主键约束(Primary Key Constraint,PK):数据唯一且不为空。
  2. 唯一约束(Unique Constraint,UQ):唯一，允许且仅有一个空。
  3. 检查约束(Check Constraint,CK):范围限制，格式限制。如年龄有关的约束。
  4. 外键约束(Foreign Key Constraint,FK):两表建立联系。

    语法：
    添加约束：alter table 表名 add constraint 约束名 约束类型 具体的约束说明
    删除约束：add 变成 drop

数据管理(DML语言)

插入数据：insert into 表名 (column) values (对应的数据);
更新数据：update 表名 set column=value where condition;
删除语句：delete from 表名; 或者加条件：delete from 表名 where condition;

#问题/删除
drop table：删除表的全部数据和表本身。
truncate table：仅删除表的数据，释放表数据块。执行前隐式提交不可回滚。
delete from 表名:逐行删除，可回滚。

基础查询

投影查询：
==select== * ==from== 表名;—查询表所有列的数据

  • 可以为column，也可进行数学运算
    from 表名 ==where== condition； 增加查询的条件

非空问题：
如果数据为空且参与运算，结果就为空
select ename,sal,comm.sal+==NVL(comm,0)== from emp;

查询列的别名：
select ename as “…”
select ename “…”
select ename …
字符串拼接：
select ename||’…’||sal from emp;
去重：
select ==DISTINCT== job from emp;

查询结果排序：order by
select ename from emp where ename like ‘%O%’
==order by== sal ==desc==;
有==asc(升序);desc(降序)==

Where条件

  1. 简单运算(大于小于..)
  2. 其他运算符
    1. ==between…and…== 例如：select sal form emp where ==sal>=1000==;
    2. ==in(set)== 例如：select job fronm emp where job ==in(‘CLERK’,’MANAGER’)==;
    3. ==NOT== 取反
    4. ==like== 模糊查询。其中%代表任意个字符；_ 代表一个字符。
    5. ==is null== 空值
  3. 逻辑运算：and(逻辑并);or(逻辑或);not(逻辑否)
  4. 优先级：运算>连接>比较>is null like..>between>not>and>or
