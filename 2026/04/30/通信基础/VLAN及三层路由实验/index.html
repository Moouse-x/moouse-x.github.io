vlan

让两个相同的地址范围实现隔离
减少广播风暴
默认情况下所有接口都在vlan1里面
划分方法
基于端口
基于MAC地址
基于IP子网划分
基于协议划分
基于策略

端口类型
ACCESS：接口模式，直接连接终端
在接收到数据后会添加vlan tag，vlan id和端口的pvid相同
在转发数据前会移除vlan tag
配置
进入接口视图
port link-type access
port default vlan10
TRUNK：干道模式，连接交换机的端口
收到帧时，如果该帧不包含tag，则添加端口的pvid；如果包含tag，则不改变
发送帧时，该帧的vlan id在trunk上的允许发送列表中：若与端口的pvid相同时，则剥离tag发送；若与端口的pvid不同时，则直接发送
配置
在交换机之间为port link-type trunk
port trunk allow-pass vlan all
Hybrid：可以手动控制这个接口是不是要剥离或携带tag帧
发送帧时，检查该端口是否允许该valn数据帧通过，如果允许，则可以通过命令配置发送时是否携带tag帧
配置
交换机-交换机端口
port link-type hybrid
port hybrid tagged vlan 2 3 100 —打上标签
交换机-终端端口
port hybrid pvid vlan 2
port hybrid untagged vlan 2 100 —剥离标签
